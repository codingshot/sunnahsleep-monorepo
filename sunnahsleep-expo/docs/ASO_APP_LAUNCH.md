# App Launch Timeline — Sunnah Sleep

Based on [aso-skills app-launch](https://github.com/Eronred/aso-skills). Use this as the master checklist for launch.

---

## 8 weeks before launch

**ASO**
- [ ] Keyword research (see STORE_LISTING + app-marketing-context)
- [ ] Metadata: title, subtitle, keyword field, description
- [ ] Competitor analysis (see ASO_COMPETITOR_ANALYSIS.md)
- [ ] Screenshot plan: 10 slots, benefit-led (see STORE_LISTING)
- [ ] App preview video (15–30 s) optional
- [ ] Icon final

**Pre-launch page (if new app)**
- [ ] App Store pre-order page
- [ ] Pre-order link for early sign-ups

**Press & community**
- [ ] Press kit: screenshots, icon, short description, founder story
- [ ] List 20–30 relevant journalists / bloggers / YouTubers
- [ ] Draft pitch emails
- [ ] Identify communities: Reddit, Discord, Islamic app forums, Twitter/X

---

## 4 weeks before launch

**Beta & feedback**
- [ ] TestFlight beta (50–200 users)
- [ ] Fix critical issues from feedback
- [ ] Ask beta testers to be ready to review on launch day
- [ ] FAQ and support docs

**Content**
- [ ] Launch blog post
- [ ] 10+ social posts for launch week
- [ ] Demo video for social
- [ ] Product Hunt draft (if using)

**Technical**
- [ ] Analytics (see app-analytics if using)
- [ ] In-app rating prompt (after positive moment, 3+ sessions, 7+ days)
- [ ] Crash reporting
- [ ] Test IAP if any

---

## 1 week before launch

**Final prep**
- [ ] Submit for review (2–3 day buffer)
- [ ] Schedule press embargo for launch day
- [ ] Launch-day social posts ready
- [ ] Brief beta testers on review timing
- [ ] Phased release vs immediate in App Store Connect
- [ ] Apple Search Ads campaign if budget (see ua-campaign)

---

## Launch day

**Morning**
- [ ] Release (or confirm phased release)
- [ ] Send press emails
- [ ] Post on social (personal + app)
- [ ] Product Hunt if applicable
- [ ] Post in relevant communities
- [ ] Email list

**During day**
- [ ] Monitor App Store Connect
- [ ] Respond to every review (see ASO_REVIEW_MANAGEMENT.md)
- [ ] Engage with mentions
- [ ] Monitor crashes
- [ ] Share milestones (“#X in Health & Fitness!”)

**Evening**
- [ ] Thank early adopters
- [ ] Share day-one stats if strong
- [ ] Plan next day content

---

## Week 1 after launch

- [ ] Daily social posts
- [ ] Respond to all reviews
- [ ] Check keyword indexing/rankings
- [ ] Adjust Apple Search Ads if running
- [ ] Follow up with press
- [ ] Submit for App Store featuring (see ASO_FEATURED.md)
- [ ] Review first-week metrics

---

## Month 1 after launch

- [ ] Full ASO audit with real data (STORE_LISTING self-audit)
- [ ] Tweak metadata from keyword performance
- [ ] A/B test screenshots (see ASO_AB_TESTING.md)
- [ ] Plan first major update from feedback
- [ ] Evaluate paid acquisition ROI
- [ ] Ongoing keyword tracking

---

## Free channels (Sunnah Sleep)

1. **Product Hunt** — If positioning as productivity/wellness
2. **Reddit** — r/islam, r/muslims, relevant sleep/wellness subs (follow rules)
3. **Twitter/X** — Build-in-public, launch thread
4. **Islamic forums / communities** — Where audience already is
5. **App review sites** — MacStories, 9to5Mac, Islamic tech blogs

## Paid (optional)

1. **Apple Search Ads** — High intent (“fajr alarm”, “prayer times”)
2. **Meta** — If targeting Muslim demographics
3. **Micro-influencers** — Islamic wellness / productivity
